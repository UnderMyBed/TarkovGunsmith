# @tarkov/web

TarkovGunsmith SPA. Vite + React + TanStack Router/Query, deployed to Cloudflare Pages.

## Quick start

```bash
pnpm install
pnpm --filter @tarkov/web dev          # http://localhost:5173
```

Visit `/smoke` to verify the data layer is wired correctly (calls api.tarkov.dev via @tarkov/data).

See [`CLAUDE.md`](./CLAUDE.md) for conventions.
